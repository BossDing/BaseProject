# BaseProject
#项目介绍：
  本项目(Android库工程)为Android开发APP的通用模版框架，皆在使用本框架能快速开发Android项目尤其是含有网络请求的项目，优化项目代码结构、模块化编程。
本项目力求能给阅读者、使用者在Android项目架构能力方面带来一定的速度、质量的提升。
